# PART 1 — MAKE SLASH MODE MUCH BETTER

The current slash mode is too bland:
- objects slowly fall top to bottom
- slashing is too easy
- there is little tension, surprise, or strategy
- visuals feel placeholder

I want slash mode to feel exciting, readable, and addictive.

## 1A. Improve object movement and spawn design
Replace the current simple falling behavior with more varied motion patterns:
- fast drops
- slow heavy drops
- angled movement
- curved arcs
- burst spawn clusters
- staggered waves
- occasional fake-out timing
- rare high-value objects that move faster or require precision

Create spawn logic that changes during the 20-second run:
- seconds 0–5: easier warmup
- seconds 5–12: denser and more varied
- seconds 12–18: high intensity
- seconds 18–20: climax / jackpot chance / risk spike

## 1B. Improve risk/reward and decision-making
Add more meaningful object types.

### Positive objects
- generic player shard
- attacker shard
- midfielder shard
- defender shard
- trait fragment
- coins
- evolution catalyst
- golden prospect pack
- combo orb
- scout intel token
- physio token
- morale boost token

### Negative / risky objects
- red card trap
- injury hazard
- fake agent file
- burnout token
- scouting miss token

### New football-connected event objects
Add these and make them affect club state:
- injury hazard: if hit, randomly injure one lineup player for the next match or reduce that player’s condition
- burnout token: reduces manager XP or next slash reward quality
- fake agent file: converts one reward roll into low-quality currency
- morale boost token: next match gets a small team chemistry or composure boost
- physio token: removes one active injury or prevents next injury effect
- scout intel token: guarantees one role-targeted reward at end of run

These should make slash feel more connected to football club management.

## 1C. Improve combo and juice
Make combos much more satisfying:
- visible combo meter
- combo tier names like Hot, On Fire, Unstoppable
- better multiplier feedback
- particle burst on combo increase
- screen shake on rare hits
- freeze-frame hit pause on golden objects
- haptic feedback hooks where possible in Expo
- stronger sound hooks for success, danger, and jackpot moments

## 1D. Make slashing require more skill
Improve collision / gesture feel:
- make swipe trails look sharp and satisfying
- reward slicing multiple items in one swipe
- penalize sloppy swipes only lightly
- make rare objects smaller or faster
- allow “perfect chain” moments
- add near-miss tension with hazards

## 1E. Visual improvements for slash mode
Upgrade the slash assets and overall presentation:
- replace placeholder shapes with polished football-themed objects
- player shards should look like neon player-card fragments
- role shards should have football role icons
- trait fragments should have distinct symbols
- catalysts should feel rare and premium
- hazards should be instantly readable
- add depth with shadows, glow, scale, and parallax
- improve reward reveal screen so it feels premium and exciting

Use a sleek, modern, high-contrast football sci-fi aesthetic.
Do not make it childish.
